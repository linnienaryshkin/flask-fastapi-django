# flask-fastapi-django

Learning python stack

## Setup

1. Create an environment

```sh
python3 -m venv .venv
```

2. Activate the environment

```sh
. .venv/bin/activate
```

3. Install Flask

```sh
pip install Flask
```

4. Run

```sh
flask run --reload
```
